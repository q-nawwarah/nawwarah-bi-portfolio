Prepared and transformed raw financial sales data into a clean, BI-ready dataset, enabling reliable KPI reporting and dashboard development.

## Skills Demonstrated
Python (Pandas, Matplotlib) • Data Cleaning & Standardization • Feature Engineering • Exploratory Data Analysis • BI-Ready Data Preparation

## Business Impact
- Produces a trusted dataset for accurate dashboards
- Reduces downstream BI rework and ensures confident decision-making
- Supports revenue, profit, and margin analysis across time and regions

- Cleaned and standardized raw sales data
- Resolved missing values affecting profitability metrics
- Engineered key business metrics, e.g., profit margin and monthly trends

- Delivered a structured dataset ready for Power BI, Tableau, or Excel

## Outcome
- dataset_financial2014_clean.csv ready for BI reporting
