### 3. Malaysia COVID-19: Risk & Resilience Analysis
Analyzed Malaysia’s COVID-19 trajectory to identify high-risk periods and evaluate the impact of the national vaccination rollout using SQL and Power BI.

#### Skills Demonstrated: SQL (BigQuery) • Power BI • Time-Series Analysis • Risk Monitoring

#### Business Impact:
- Identifies high-risk periods for healthcare strain
- Measures vaccination effectiveness via Case Fatality Rate (CFR) trends
- Provides early-warning indicators (~14-day infection-to-death lag)
- Supports ICU capacity planning and vaccine prioritization

#### Technical Highlights:
- Modular SQL pipeline for validation, aggregation, peak-period flagging, and lag analysis
- Advanced SQL techniques: window functions, safe arithmetic, reusable queries
- SQL outputs connected to Power BI dashboards for executive monitoring

#### Key Insights:
- Peak healthcare strain: August 2021 (Delta wave)
- CFR remained lower post-vaccination, showing program impact
- Infection-to-death lag enables proactive capacity planning
- Regional benchmarking highlights higher vaccination coverage vs neighbors

#### Outcome:
- BI-ready dashboard for healthcare decision-makers
- Provides actionable insights for ICU capacity, vaccination strategy, and risk monitoring

#### Dataset: Our World in Data COVID-19 (2020–2021, Malaysia)
